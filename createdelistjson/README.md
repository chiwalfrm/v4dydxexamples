**These scripts create the json file to delist markets.**<br>
Put all files in a directory and then run delistmarkets.sh.
<br>
<br>
Syntax: ```./delistmarkets.sh <market1> <market2> ...```<br>
Example: ```./delistmarkets.sh TIBBIR-USD DAKU-USD```
